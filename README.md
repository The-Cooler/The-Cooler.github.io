# Java学习
## 基础语法
*变量类型
> int 整数型
> double 浮点型
> char 字符型
> boolean 布尔型

<u>注意</u>：Java中的逻辑符号只有true和false,不能用1和0表示。

*运算符
> + 加法
> - 减法
> * 乘法
> / 除法
> % 取余
> ++ 自增
> -- 自减
> = 赋值
> == 等于
>!= 不等于
> > 大于
> < 小于
> >= 大于等于
> <= 小于等于

*控制语句
> if 条件语句
> switch 多分支语句
> for 循环语句
> while 循环语句
> do-while 循环语句
